# Tablet Holder Project

Параметрическая модель держателя для планшета с креплением VESA 100.

## Архитектура модели

После рефакторинга модель следует принципам **надежного CAD-моделирования**:

1. **Абсолютные координаты (Locations)**: Все детали строятся относительно своих локальных `Locations`, а не через автоматическое центрирование (`Align.CENTER`). Это исключает накопление микро-ошибок при дробных размерах.
2. **Плоскость стыка (X=0)**: Стык половин всегда находится в нулевой плоскости X. Это гарантирует идеальное прилегание.
3. **Фиксация базы в сборке**: В `assembly.py` основные детали (`lp`, `rp`) принудительно фиксируются в `Location((0,0,0))`. Любые дополнительные детали (адаптеры, крепления) позиционируются относительно этой базы через явный расчет трансформаций джойнтов.
4. **Отказ от неявных зависимостей**: Мы не используем `connect_to` для основных половин, так как в старых версиях `build123d` это может приводить к нежелательному смещению родительских объектов.

## Параметры (params.json)

- `tablet_w/h/t`: Габариты планшета (с учетом чехла).
- `wall`: Толщина стенок (рекомендуется 8мм для жесткости).
- `visor_angle/d`: Угол и вылет козырька.
- `adapter_w`: Размер VESA площадки.

## Сборка

Для сборки используется класс `ProjectAssembly`, который возвращает единый `Compound` объект, готовый к рендерингу или экспорту.
